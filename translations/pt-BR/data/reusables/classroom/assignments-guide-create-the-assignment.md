#### Se não houver recomendações na sala de aula

To create your first assignment:

1. Efetue o login em {% data variables.product.prodname_classroom_with_url %}.
1. Acesse uma sala de aula.
1. In the {% octicon "repo" aria-label="The repo icon" %} **Assignments** tab, click **Create an assignment**.

#### Se já existirem recomendações na sala de aula

To create an assignment:

1. Efetue o login em {% data variables.product.prodname_classroom_with_url %}.
1. Acesse uma sala de aula.
1. Na aba de {% octicon "repo" aria-label="The repo icon" %} **Atividades**, clique em **Nova atividade**.
