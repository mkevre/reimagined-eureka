マシンタイプを切り替えることで、作業に最も適したリソースを柔軟に選択できます。 特待のタスクでは、様々なマシンタイプが必要になることがあります。

* ファイルの更新とPull Requestの作成に対しては**Basic**
* Webサイトの編集に対しては**Standard**
* データ分析のような高負荷のコンピューティングタスクに対しては**Premium**
